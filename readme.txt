Create tiled sliders with various eye-catching sliding effects

== Description ==

Tired of standard sliders? Roll your own!

Build your own slideshow and put together highly customizable tiles by a simple drag and drop interface.

The plugin ships with its own post type for sliding content.

It's simple: create some posts, add a featured image (if any) and choose one or more of the existing tile types.
The plugin will then display the HTML and images you posted and display the way you like.

See the [Demo](http://crosstec.de/wp-tilebuilder-demo/)

[Documentation](http://crosstec.de/media/wp-tilebuilder-docs/)

*** Tutorial Video ***

[youtube https://www.youtube.com/watch?v=ZBqhS1nELYg]

Features of the plugin include:

* 3 different tile types with plenty of options to customize
* Various sliding and transition effects
* Highly intuitive, drag-and-drop user interface
* Integrate seamlessly with WordPress admin interface
* Compatible with WordPress Import/Exporter
* Unlimited number of reusable slideshow templates
* Highly extensible
* Developer friendly with dozens of actions and filters

More information at [Crosstec.de](http://crosstec.de/).

== Installation ==

1. Activate the plugin
2. Go to Tile Builder and create new tiles to create a slideshow template.
3. To display your slider, create a post or page and click on "Add Template" to add the shortcode.
4. For detailed instructions please refer to [Crosstec](http://crosstec.de/).
